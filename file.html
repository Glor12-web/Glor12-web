JavaScript is a high-level, interpreted programming language primarily used for web development. Here are some basics: 1. **Variables**: Used to store data. ```javascript let name = "Alice"; // ES6 variable declaration const age = 25; // Constant variable var city = "London"; // Older variable declaration ``` 2. **Data Types**: Common types include: - String: `"Hello"` - Number: `123` - Boolean: `true` or `false` - Object: `{ key: "value" }` - Array: `[1, 2, 3]` - Undefined: `undefined` - Null: `null` 3. **Functions**: Blocks of code designed to perform a task. ```javascript function greet() { console.log("Hello, World!"); } greet(); ``` 4. **Control Structures**: Conditional statements and loops. ```javascript if (age > 18) { console.log("Adult"); } else { console.log("Minor"); } for (let i = 0; i < 5; i++) { console.log(i); } ``` 5. **Events**: JavaScript can respond to user interactions. ```javascript document.getElementById("myButton").onclick = function() { alert("Button clicked!"); }; ``` 6. **DOM Manipulation**: You can modify the HTML structure. ```javascript document.getElementById("myElement").innerHTML = "New Content"; ``` 7. **ES6 Features**: Introduced modern syntax and features. - Arrow Functions: ```javascript const add = (a, b) => a + b; ``` - Template Literals: ```javascript const greeting = `Hello, ${name}`; ```
